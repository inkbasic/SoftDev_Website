# SoftDev Website 🚀

A modern React-based travel platform for managing locations, bookings, and user profiles with a beautiful UI built using Tailwind CSS and shadcn/ui components.


## 🛠️ Tech Stack

- **Frontend Framework**: React 19.1.1
- **Routing**: React Router DOM v7.8.2
- **Styling**: 
  - Tailwind CSS v4.1.13
  - Custom CSS variables for theming
- **UI Components**: 
  - shadcn/ui components
  - Radix UI primitives
  - Lucide React icons
- **Charts**: Recharts v2.15.4
- **Build Tool**: Vite v7.1.2
- **Package Manager**: npm

## 🚀 Getting Started

### Prerequisites

- Node.js (version 18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd SoftDev_Website
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

## 📝 Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run lint` - Run ESLint
- `npm run preview` - Preview production build
